# A-Maze-ing

*This project has been created as part of the 42 curriculum by imoulasr.*

## Description

A Python maze generator that creates perfect mazes with configurable dimensions, 
entry/exit points, and visual representation. The generator uses the Recursive 
Backtracker algorithm to ensure a single valid path between any two points.

## Instructions

### Installation
\`\`\`bash
make install
\`\`\`

### Running
\`\`\`bash
make run
\`\`\`

### Configuration
Edit `config.txt` to customize maze parameters.
